# Tests for [Swag Labs](www.saucedemo.com)

This is the task  of the Become a QA Automation course.

---

### Run the tests:
* Run all the tests
```
npm run test
```
* Login test
```
npm run login
```
* Inventory test
```
npm run inventory
```
* Cart test
```
npm run cart
```
* End to End test
```
npm run e2e
```
